## 2. Typography

| Role | Font | Weight | Use Cases |
|------|------|--------|-----------|
| **Heading** | Playfair Display | 700/900 | Titles, keywords, prize amount |
| **Body** | Inter | 300–700 | UI text, buttons, descriptions |
| **Mono** | Space Mono | 700 | TikTok logo only |

### Font Pairing Rationale
- **Playfair Display**: Elegant serif for luxury/premium feel → perfect for prize/gold theme
- **Inter** (upgraded from DM Sans): Better readability, wider weight range, friendly + modern

### Size Scale (clamp for responsive)
- H1: `clamp(28px, 8vw, 36px)`
- Prize: `clamp(40px, 12vw, 50px)`
- Body: `16px` minimum
- Small: `13px`
- Caption: `11px`
- Micro: `9px`

## 6. Motion & Animation

### Transition Durations
| Token | Value | Usage |
|-------|-------|-------|
| `--t-fast` | 150ms ease | Hover states, button press |
| `--t-base` | 200ms ease | Color changes, background |
| `--t-slow` | 300ms ease | Opacity, transform |
| `--t-spring` | 300ms spring | Modal entrance |

### Key Animations
- Card float: Y ±8px, 3s ease-in-out infinite
- Card flip: rotateY(180deg), 0.8s cubic-bezier
- Button shimmer: 2.5s linear sweep
- Snowfall: 5–11s linear
- Confetti: 2–4s ease-out

### Reduced Motion
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 8. Accessibility Checklist

- [x] All interactive elements have `cursor: pointer`
- [x] Minimum touch target: 44×44px
- [x] Focus-visible outline on all focusable elements
- [x] ARIA labels on buttons, dialogs, navigation
- [x] `role="dialog"` + `aria-modal="true"` on overlays
- [x] `role="list"` / `role="listitem"` on card grid
- [x] `role="status"` + `aria-live="polite"` on toast
- [x] Keyboard navigation: ESC closes modals/overlays
- [x] Keyboard navigation: Enter/Space on flip cards
- [x] `prefers-reduced-motion` respected
- [x] No emoji used as icons (all SVG)
- [x] Text contrast ≥ 4.5:1 minimum
- [x] Semantic HTML: `<header>`, `<nav>`, `<section>`, `<main>`
- [x] Status bar decorative content has `aria-hidden="true"`

## 9. Changes from Original

| Aspect | Before | After | Rationale |
|--------|--------|-------|-----------|
| Body Font | DM Sans | **Inter** | Pro Max "Classic Elegant" pairing |
| Icons | Emoji (🎁🔔✨) | **Inline SVG** | Pro Max rule: no emoji as UI icons |
| Text opacity | 0.55 / 0.3 | **0.65 / 0.4** | Improved WCAG contrast |
| Touch targets | Mixed sizes | **44px minimum** | Pro Max touch target rule |
| Focus states | None | **2px gold outline** | Keyboard accessibility |
| Reduced motion | Not handled | **Media query** | Animation accessibility |
| Z-index | Random values | **10/20/30/40/50 scale** | Pro Max z-index management |
| Transitions | Mixed timings | **150/200/300ms system** | Consistent micro-interactions |
| Spacing | Magic numbers | **8px grid tokens** | Design system consistency |
| Radius | Magic numbers | **Token system** | Consistent radius scale |
| Keyboard nav | None | **ESC + Enter/Space** | Full keyboard support |
| ARIA | Minimal | **Full coverage** | Screen reader support |
| Responsive | Fixed values | **clamp() + fluid** | Better scaling |
| Container | 750px max | **430px max** | Mobile-first H5 |
| Hover states | Basic | **Smooth + visual feedback** | Better interaction cues |
